# @lalalai/enjin

## 0.1.0

### Minor Changes

- Initial release of LalalAI Enjin, an AI-powered YouTube video clipper.

  New Features:
  - Automatic clip generation from YouTube URLs using AI analysis
  - Video transcription with Whisper
  - Clip slicing and storage with progress tracking
  - Frontend UI for clip management and video preview
  - Remotion integration for captioned video rendering

  Improvements:
  - Full workflow implementation with Convex backend
  - Database schema for clips and files
  - UI components and styling updates

  This marks the first major version with complete video processing functionality.
